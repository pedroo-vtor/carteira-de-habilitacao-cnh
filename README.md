# Carteira Nacional de Habilitação (CNH)

## Informações
- Tarefa de casa da disciplina de Framework Back-End.
- API simples em Flask, seguindo o padrão MVC, para gerenciar carteiras de habilitação, permitindo criar, listar, atualizar e excluir registros (CRUD completo).

## Rotas

- **Raíz**: `http://localhost:5000`
- **Create:** `http://localhost:5000/carteiras`  
- **Read:** `http://localhost:5000/carteiras`  
- **Update:** `http://localhost:5000/carteiras/<int:carteira_id>`  
- **Delete:** `http://localhost:5000/carteiras/<int:carteira_id>`
